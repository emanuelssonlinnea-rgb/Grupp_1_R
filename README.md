# E-handel – analysera försäljning, kundbeteende och returmönster

Vi arbetar som ett analysteam på ett e-handelsföretag som säljer produkter inom flera kategorier till kunder i olika regioner.
Företaget samlar in data om bland annat: ordrar, kunder, produktkategorier, rabatter, leveranstider, returer.



## Filer

-   `Grupp_1_Dataförståelse.R`läser in data, EDA - dataförståelse
-   `data_städning_förberedesle.R`städar och förbereder data
-   `Statistiska_sammanfattningar.R`gör analysen
-   `Gruppuppgift_Visualisering.R`skapar och sparar figuren
-   `run_analysis.R`kör hela projektet



## Så kör du projektet

Öppna projektet i RStudio och kör `run_analysis.R`

## Uppdrag

Ledningen vill få bättre förståelse för verksamheten och vill använda data som underlag för beslut.
De vill till exempel förstå:

- vilka kategorier som driver försäljning

- om vissa kundgrupper beter sig annorlunda än andra

- om rabatter verkar påverka ordervärde eller returgrad

- om det finns regioner eller kategorier med särskilda mönster

- om leveranstid verkar ha samband med returer

Målet är inte att bygga en avancerad modell.
Målet är att göra en tydlig och relevant dataanalys i R och presentera insikter på ett sätt som går att förstå och använda.
